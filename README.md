# DocExtractify — Always Offline. Always Private.

A native Windows desktop application built with **WinUI 3 (Windows App SDK)** that converts **PDF, Word, PowerPoint, Excel, CSV, and images** into **Markdown, plain text, or Word (.docx)**. Packaged as **MSIX** for deployment via the Microsoft Store or direct install.

All processing happens locally — no cloud, no telemetry, no data leaves your device.

The app auto-starts a local Python backend and provides a Fluent Design UI with drag-and-drop, batch conversion, history, and dark/light theme support.

---

## Supported Formats

### Input

| Category    | Extensions |
|-------------|-----------|
| PDF         | `.pdf` |
| Word        | `.docx` `.docm` |
| PowerPoint  | `.pptx` |
| Excel / CSV | `.xlsx` `.xls` `.xlsm` `.csv` |
| Images      | `.png` `.jpg` `.jpeg` `.gif` `.bmp` `.tiff` `.tif` `.webp` `.wmf` `.emf` |

### Output

| Format | Extension | Notes |
|--------|-----------|-------|
| `md`   | `.md`     | Markdown — default, always generated |
| `txt`  | `.txt`    | Plain text — Markdown stripped |
| `docx` | `.docx`   | Word — headings, tables, bold/italic preserved |

## Setup

### 1. Install Python dependencies

```powershell
python -m venv .venv
.venv\Scripts\activate

pip install -r requirements.txt
```

### 2. Generate logo assets

```powershell
.\DocumentConverterWinUI\Generate-Assets.ps1
```

### 3. Build & Run

Open `DocumentConverterWinUI\DocumentConverter.csproj` in **Visual Studio 2022** and press **F5**.

The app will auto-start the Python backend (`api.py`) on port **8080**.

See **[DocumentConverterWinUI/README-WinUI.md](DocumentConverterWinUI/README-WinUI.md)** for the full build, packaging, and MSIX deployment guide.

## Privacy & Security

> **Always Offline. Always Private.**

No telemetry. No remote crash reports. No cookies. All processing happens
locally. To eliminate residual outbound calls (model downloads, framework
version checks):

```powershell
$env:HF_HUB_OFFLINE = "1"
$env:TRANSFORMERS_OFFLINE = "1"
python api.py
```
